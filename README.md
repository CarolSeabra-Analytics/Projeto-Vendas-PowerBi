📊 Projeto de Análise de Vendas
Este projeto foi desenvolvido com o objetivo de analisar dados de vendas e gerar insights de negócio utilizando SQL e Power BI.

🔍 Análises realizadas
Faturamento total da empresa
Vendas por categoria de produto
Vendas por região
Ranking de clientes com maior faturamento

📈 Principais insights
A categoria Eletrônicos apresenta o maior volume de vendas
A região Sudeste concentra a maior parte do faturamento
Alguns clientes se destacam com alto valor de compra, indicando potencial de fidelização

🛠️ Ferramentas utilizadas
SQL (análise e consultas)
Power BI (visualização de dados)
Google Sheets (tratamento de dados)

Dashboard em Power BI (.pbix)
